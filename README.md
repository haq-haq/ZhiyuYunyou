# 智渔云游 · 数字平台

北京市密云区溪翁庄镇走马庄村农文旅融合数字平台，围绕密云水库渔文化，构建渔食、渔居、渔赏、渔享一体化体验体系。

## 快速启动

```bash
npm install
npm run dev      # http://localhost:5173
npm run build
npm run preview
```

## 项目定位

- **地点**：北京市密云区溪翁庄镇走马庄村
- **核心**：密云水库渔文化
- **体系**：渔食 · 渔居 · 渔赏 · 渔享
- **目标**：农业→体验、文化→消费、生态→经济，助力产业增值、农民增收、乡村振兴
